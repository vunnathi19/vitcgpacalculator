📘 VIT GPA & CGPA Adviser System

A smart academic performance tracker that calculates GPA and CGPA, analyzes grade history from screenshots, and provides improvement suggestions using an automated adviser system.

🚀 Features
📊 GPA Calculator
Add multiple subjects
Enter credits and grades (S–F)
Auto-calculates:
Grade Points
Weighted GPA
📈 CGPA Calculator
Add semester-wise data
Enter GPA + credits per semester
Computes overall CGPA using weighted formula
🧠 AI Adviser (OCR Analysis)
Upload grade history screenshot
Extracts grades using OCR (Tesseract.js)
Identifies weakest 3 subjects
Suggests improvement strategy
Simulates CGPA improvement scenarios
🔄 CGPA Simulation Engine
Shows:
Current CGPA
Subject-wise improvement impact
Best-case CGPA (after improving weakest subjects)
⚙️ Tech Stack
HTML5
CSS3
JavaScript (Vanilla)
Tesseract.js (OCR engine)
📌 How It Works
Enter GPA or CGPA data manually OR
Upload grade screenshot
System extracts grades automatically
Weak subjects are identified
Improvement cases are simulated
Final CGPA improvement is displayed
